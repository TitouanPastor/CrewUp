# GDPR Compliance

CrewUp processes personal data of EU users and must comply with GDPR.

## Ongoing Work

We are actively working on GDPR compliance. For detailed status, implementation plans, and tracked issues, see:

**GitHub Issues**: https://github.com/TitouanPastor/CrewUp/issues

Relevant issues:
- Data export functionality (Art. 15 - Right to Access)
- Account deletion (Art. 17 - Right to Erasure)  
- Privacy policy and consent flows (Art. 12-13)
- Age verification (Art. 8)

## Current Implementation

**Security Measures** (Art. 32):
- TLS 1.3 encryption
- Keycloak authentication
- Database access controls
- JWT token expiration

**Data Protection by Design** (Art. 25):
- UUID pseudonymization
- Minimal data collection
- Private event settings
